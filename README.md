# SAKAMOMO FAMILY SERVICE

AI（LLMなど）を利用したサービス（バックエンド、フロントエンド含む）を本リポジトリで統合的に構築していきます.<br>
※作成者が使いたい機能や勉強したい機能を実装中となります.

## フォルダ構成

| フォルダ名 | 概要 |
| ---- | ---- |
| backend | サービスのバックエンド部分のコード、ビルド手順などをまとめています |
| frontend | サービスのフロントエンド部分のコード、ビルド手順などをまとめています |
| pipeline | サービスで利用するデータをAPIとは別で定期的に構築するためのパイプライン処理に関するコード、ビルド手順をまとめています |
| sample_codes | backend, frontend, pipelineを構築する際に実験的に構築したコードをまとめています |

## セットアップ

上記各フォルダのドキュメントを参照して、インフラを構築する.<br>
現状は一部google cloud console上での手動構築が必要となる.
